# Atrofiled-by-p7

Astrophotography datasets and processing examples for learning deep sky image stacking with Siril.

This repository contains astrophotography datasets and processing examples for learning image stacking and shooting techniques.
The files help you practice stacking, calibration, and basic processing using Siril or Sequator.

The dataset includes

light frames
dark frames
flat frames
bias frames

"no need for bias if u got large sensor" :)

These files allow you to test a full calibration and stacking workflow.

The main goal here is to show how deep sky images can be captured and processed with simple equipment.
All example images were taken using any DSLR with basic lenses ofc..

what u can find here is :

correct stacking workflow
camera settings for astrophotography
calibration frame usage
basic processing steps


This project is useful for beginners learning astrophotography and ppl practicing image stacking ... and testing scripts for Siril

Example workflow

Capture light frames of the target object.
Capture calibration frames dark flat bias.
Stack the frames in Siril.
Perform final adjustments in Lr mobile or pc
